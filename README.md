 **DESARROLLO**
1.	La APP es un conjunto de programas que está orientado al usuario final para el móvil. Específicamente de software diseñado para ejecutar en dispositivos móviles, celular, tablets 
2.	Frontend es una parte de la aplicación con el usuario interactúa directamente (La interfaz. Los botones, colores, pantallas). Se encarga de la experiencia del usuario (UX) y el diseño (UI). Backend es el detrás de escena, este se encarga de la lógica, el procesamiento de datos, la seguridad y la conexión con la base de datos. Se comunican mediante peticiones HTTP. El Frontend "dame los datos del usuario" y backend responde con esos datos.
3.  API REST es un conjunto de reglas y estándares que permiten que fronted y backend de comuniquen entre si atreves de la web.
4. JDON (java script objet notation) es un formato ligero para el intercambio de datos, fácil de leer para humanos y fácil de procesar para máquinas.

7. La programación orientada a objeto(POO) es un paradigma de programación que organiza el diseño de software alrededor de “objetos” (datos y comportamiento) en lugar de funciones y lógica, modelando entidades del mundo real. Facilita la reutilización, estructura y mantenimiento de código complejo mediante clases, encapsulamiento, herencia y polimorfismo.
CLASES: es una planilla o modelo para crear objetos, de la cual define los atributos(datos) y métodos(acciones) que tendrán los objetos
OBJETO: una instancia concreta de una clase (ej. un objeto “coche” especifico basado en la clase “vehículo”).
HERENCIA: la propiedad que tenga el objeto implemento métodos de su objeto que llama clase.

8. Public: el atributo o método es accesible desde cualquier lugar. Cualquier otra parte del programa puede leerlo o cambiarlo directamente 
Prívate: es una clase distintas no puede modificar un objeto a otro
